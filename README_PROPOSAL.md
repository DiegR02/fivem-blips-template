## Overview

`fivem-blips-template` is a Lua-based template for adding and managing map blips
in [FiveM](https://fivem.net/) servers. It provides a structured starting point
for developers who want to display custom locations, markers, or points of
interest on the in-game minimap and main map.

## Installation

> ⚠️ Review the repository source files to confirm exact setup requirements
> before following these steps.

1. Clone or download this repository into your FiveM server `resources` folder:
   ```bash
   cd /path/to/server/resources
   git clone https://github.com/DiegR02/Fivem_BlipsTemplate.git fivem-blips-template
   ```
2. Add the resource to your `server.cfg`:
   ```
   ensure fivem-blips-template
   ```
3. Restart the server or use `refresh` followed by `ensure fivem-blips-template`
   in the server console.

## Usage

> ⚠️ Validate the project entrypoint before relying on these examples.

- Open the main Lua script and locate the blips configuration table.
- Add or modify entries to define new blip locations, sprites, colors, and
  labels.
- Save the file and restart the resource to see changes reflected in-game.
